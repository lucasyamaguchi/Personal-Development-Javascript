Para a criação deste espaço e centralização dos conteúdos de javascript, foi utilizado o github e o codespaces. Ao abrir um novo repositório e iniciar as anotações, foi então criada a estrutura básica do projeto, as dependências e licenças. Instalação do Node: nvm --v nvm install lts/hydrogen nvm alias default lts/hydrogen Criação do arquivo .nvmrc, que indica ao diretório qual versão do node usaremos. lts/hydrogen npm init Por se tratar de um repositório no Github, a fim de executar o arquivo da aula para ver as atualizações, utilizei a extensão Live-server, esta que foi instalada através do terminal: npm install live-server -D Após as instalações, é selecionada a formatação do código, que será realizada através do Prettier, este, que também possui instalação através do node. npm install prettier -D Após instalação dos componentes básicos para o repositório funcionar como esperado, foram criados os arquivos de configurações com as seguintes instruções: .gitignore |-> node_modules .editorconfig |-> root = true [*] indent_style = space indent_size = 2 Por fim, para facilitar o uso recorrente de alguns comandos, são criados scripts no arquivo package.json: "scripts": { "lint:check": "prettier --check .", "lint:fix": "prettier --wtrite .", "live-server": "npx live-server" } Sendo este último, o responsável por rodar o live server do repositório, fazendo assim com que possamos selecionar qual arquivo .html com os códigos em .js da aula queremos rodar, sem necessidade de utilizar o Next na aplicação.
